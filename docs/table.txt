# Experiment Targets Table

This file defines reference targets for experiment families. Keep the schema
stable so a coding agent can read and update it safely.

## Column Definitions

- `family`: model family or architecture family
- `dataset`: dataset used for the run
- `model_size`: parameter scale label
- `tokens_b`: training tokens in billions
- `train_target`: expected train-side target metric
- `test_target`: expected test-side target metric
- `runtime_hours`: expected runtime in hours
- `notes`: optional remarks
